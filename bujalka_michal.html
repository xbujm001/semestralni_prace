Úvod

XML soubor slouží k nastavení chytré televize Kuki. Jednotlivé složky, byly převzány z prostředí Kuki TV a nastavení samotné televize. Projekt se skládá ze souborů XML (samotné nastavení televize), XSD (schéma konfigurace), XSL (potřebný pro transformaci do JSON), JSON a HTML5 (dokumentace).


Metodika

Programy a pomůcky pro vytvoření tohoto projektu byly:

  • Notepad++ a Plugin "XML Tools"- použit na psaní XML,XSD,XSL a HTML5 souborů a pro provedení transformace
  • Porovnání správnosti XML a XSD projekt
  • Pomoc při řešní problémů
  • Validátor pro ověření HTML5

Popis transformace

Transformace nejdříve vypíše informace o kontu uživatele, tzn. jméno, příjmení, datum narozeni, adresa, ID Kuki a tarif uživatele. Následně vypíše ke každému zařízení jeho číslo ID a následně všechny pozice kanálu. U pozic kanálu vypíše zda je kanál zamčený a zda se náchází v seznamu oblíbených kanálů.


Závěr

S XML jsme se setkali již v prvním ročníku a to čtení a ukládání z programů. Nyní jsem si vyzkoušel vlastní tvorbu XML konfigurace, vytvoření XML schéma a následně její transformace do JSON. Veškérý postup a detaily tohoto semestrální projektu byly zapsány do této dokumentace napsané v HTML5.
